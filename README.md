# mbot
My first Arduino project built with makeblock bot 

## Usage

1\. Start

```sh
npm start
```

2\. Use `up` & `down` to iterate through PINs

3\. Use `left` & `right` to assin `high` & `low`

## 红外

D2 - 红外接受
D3 - 红外发送

## 光线

D10 - 光线传感器

## 轮子
D4 - 左轮方向
D5 - 左轮
D6 - 右轮
D7 - 右轮方向

## 蜂鸣器
D8 - 蜂鸣器

## 超声测距
D12 - 超声

## LED
D13 - LED

## 传感器
A0(D14) - 左轨道传感器
A1(D15) - 右轨道传感器
